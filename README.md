# SlimeGame
2021 Game Development Final Project

## Description
2D Platformer puzzle game. Basic levels and mechanics. Main character is a slime on an adventure to take back the stolen coins.

### Mechanics
- Jumping
- Moving boxes
- Buttons
- Coin Collecting
- Keys
